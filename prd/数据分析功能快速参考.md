# 管理员数据分析功能 - 快速参考

## 🎯 核心价值

帮助管理员通过数据洞察：
- 了解员工普遍关注的问题
- 识别组织层面的共性挑战  
- 发现培训和发展机会
- 追踪组织健康度变化
- 支持数据驱动决策

## 📊 8大功能模块

### 1. 数据概览仪表板
```
总用户数 | 活跃用户 | 总对话数 | 总消息数
用户增长趋势 | 对话数量趋势 | 活跃度变化
```

### 2. 问题聚类分析
```
热门话题词云 → 职业发展、团队管理、工作压力...
问题分类统计 → 按场景/角色/业务线/GROW阶段
主题聚类分析 → 自动识别3-5个主要问题聚类
```

### 3. 对话质量分析
```
对话深度分布 → 浅层/中等/深度
GROW阶段分布 → Goal/Reality/Options/Will
用户满意度 → NPS得分、反馈关键词
```

### 4. 用户画像分析
```
角色分布 → 管理者/专家/执行者
业务线分布 → 技术/产品/运营...
工作风格分析 → 常见风格组合
用户活跃度分层 → 高/中/低活跃
```

### 5. 趋势与预测
```
时间趋势分析 → 本周热门话题变化
季节性模式 → Q1-Q4 不同关注点
预警机制 → 风险预警 + 机会识别
```

### 6. 深度洞察
```
关联分析 → 问题关联网络
用户旅程分析 → 典型路径 + 流失点
成功案例识别 → 高价值对话特征
```

### 7. 组织健康度指标
```
健康度评分 → 综合评分 0-100
部门对比 → 各部门健康度对比
改进建议 → 基于数据的行动建议
```

### 8. 可视化报告
```
自动生成报告 → 月度组织洞察报告
导出功能 → PDF/Excel/PPT
分享功能 → 链接分享 + 权限控制
```

## 🤖 AI 分析能力

### 关键词提取
```typescript
输入: 对话内容
输出: ["领导力转型", "团队激励", "绩效管理"]
```

### 问题分类
```typescript
输入: 问题描述
输出: {
  category: "团队管理",
  confidence: 0.95
}
```

### 情感分析
```typescript
输入: 对话内容
输出: {
  sentiment: "negative",
  score: 0.7,
  keywords: ["压力", "焦虑", "困惑"]
}
```

### 主题聚类
```typescript
输入: 多个对话主题
输出: [
  {
    name: "领导力转型挑战",
    description: "从技术专家到管理者...",
    percentage: 23
  }
]
```

## 🔒 隐私和合规

### 数据脱敏
- 展示聚合数据，不显示个人对话
- 使用匿名化标识
- 关键词提取时过滤敏感信息

### 权限控制
- 只有管理员可访问
- 支持角色权限细分
- 所有操作记录日志

### 合规要求
- 符合数据保护法规
- 用户知情同意
- 支持数据删除
