# 场景区分度对比分析

本文档展示"工作难题"和"职业发展"两个场景在优化后的差异性。

---

## 📊 核心特征对比

| 维度 | 工作难题 (work_problem) | 职业发展 (career_development) |
|------|------------------------|------------------------------|
| **时间视角** | 短期（24-48小时到2周） | 长期（1-3年甚至5年） |
| **核心关注** | 解决具体问题、缓解困境 | 成长路径、职业规划 |
| **问题紧迫性** | 高（影响当前工作） | 中低（探索性思考） |
| **目标类型** | 具体、可量化、短期见效 | 价值驱动、长远愿景 |
| **资源视角** | 现有资源、短期可用 | 能力建设、长期投入 |
| **行动节奏** | 24-48小时内启动 | 阶段性里程碑（月/季度） |

---

## 🎯 Goal阶段差异

### 工作难题场景

**时间框架强调**：
- "下次会议"、"接下来两周内"、"本周内"
- 避免过于长远的规划

**量化指标**：
- 具体场景描述（"会议中发言被采纳"、"每天准时下班"）
- 评分法："从1-10分，你希望提升到几分？"

**典型问题**：
- "关于这个工作难题，你希望达成的理想结果是什么？"
- "你希望在什么时间内（本周/下次会议/近期）看到进展？"

### 职业发展场景

**价值观深挖**：
- "这对你意味着什么？"
- "为什么这个目标对你重要？"

**长期愿景导向**：
- "你希望在未来3-5年成为怎样的职业人？"
- "理想的职业状态是什么样的？"

**典型问题**：
- "这个职业目标对你的意义是什么？为什么它对你重要？"
- "你希望在职业上留下什么样的印记？"

---

## 🔍 Reality阶段差异

### 工作难题场景

**聚焦可控因素**：
- 重点问"你尝试过什么"、"你是如何做的"
- 识别用户行为模式（而非抱怨他人或环境）

**挖掘短期资源**：
- "你已经在用什么方法？"
- "谁可以在短期内协助你？"
- "为什么是4分而不是2分？哪些做得还不错？"

**典型问题**：
- "你目前尝试过哪些解决方法？效果如何？"
- "在你的团队/组织内，有谁或什么资源可以帮助你？"

### 职业发展场景

**内在动机挖掘**：
- "在你做这些事情时，你最享受的是什么？"
- "是什么驱动你做出这个职业选择？"

**核心优势识别**：
- "在你过往最有成就感的时刻，你做了什么？"
- "你认为自己最核心的3个优势是什么？"

**典型问题**：
- "回顾过往，哪些经历让你特别有成就感？为什么？"
- "你的核心驱动力是什么？是成就感、影响力、还是稳定？"

---

## 💡 Options阶段差异

### 工作难题场景

**提供实用工具和方法**：
- 沟通类：PREP原则、金字塔原理、积极倾听
- 时间管理：时间四象限法、番茄工作法、每日三件事
- 团队管理：一对一谈话、SMART目标、及时反馈

**强调可操作性**：
- 每个方案都能"明天就开始"
- 无需额外审批或资源

**典型问题**：
- "有哪些具体的工具或方法可以帮到你？"
- "这些方案中，哪个最容易开始且最可能见效？"

### 职业发展场景

**职业可能性拓展**：
- "除了成为XX，你觉得还有哪些职业可能性？"
- 激发创造性思维："如果一切顺利，你会在哪条职业路径上？"

**能力发展路径**：
- "要实现职业目标，你需要发展哪些新能力？"
- "可以通过什么方式学习这些能力？"

**典型问题**：
- "基于我们的讨论，你看到了哪些职业发展的可能性？"
- "有哪些人、资源或机会可以支持你的职业发展？"

---

## ⚡ Will阶段差异

### 工作难题场景

**24-48小时行动强调**：
- "接下来24小时内"或"48小时内"
- 避免"下周"、"以后"等模糊时间

**数字化承诺**：
- "至少进行5次口头表扬"
- "每天列出3件最重要的事"

**场景化细节**：
- 不只是"提升沟通"
- 而是"下次周会用PREP原则发言：先说观点，再给3条理由"

**典型问题**：
- "你打算在接下来的24-48小时内采取什么第一步行动？"
- "具体来说，你会在什么时间、什么场合做什么？"

### 职业发展场景

**阶段性里程碑**：
- "接下来的1个月内，你可以完成什么准备工作？"
- "3个月后，你希望在职业发展上达到什么阶段？"

**长期规划意识**：
- "你打算如何持续跟进这个职业发展计划？"
- "多久之后，你想再评估一次进展？"

**典型问题**：
- "为了实现职业目标，你的第一步是什么？什么时候开始？"
- "你打算如何平衡当前工作和职业发展的投入？"

## 📝 开场白对比示例

### 工作难题场景开场白

```
你好！我是你的AI教练伙伴。我看到你提到了【如何提升团队协作效率】，
这确实是很多人会面临的挑战。

我们可以从以下几个方向来探讨：
1. 具体的工作任务执行或效率问题
2. 团队协作和人际关系挑战
3. 时间管理或工作压力问题

你最想从哪个方向开始？或者，你可以先跟我聊聊最近让你最困扰的具体情况。
```

**特点**：
- 引用"工作挑战"字段
- 提供具体工作场景的3个方向
- 邀请分享"最困扰的具体情况"（问题导向）

### 职业发展场景开场白

```
你好！我是你的AI教练伙伴。很高兴能够陪伴你探索职业发展的方向。
我看到你的发展目标是【成为技术管理者】，这很棒！

我们可以从以下几个方向来深入思考：
1. 明确1-3年的具体职业规划和路径
2. 职业转型或方向选择的探索
3. 关键能力提升和学习发展

你最想从哪个方向开始？或者，你可以先分享一下你目前在职业发展上的思考。
```

**特点**：
- 引用"发展目标"字段
- 提供职业发展相关的3个方向
- 邀请分享"职业发展上的思考"（探索导向）

## 🧪 测试建议

### 测试用例1：工作难题场景

**用户画像**：
- 角色：核心骨干/资深专家
- 业务线：技术/研发
- 工作挑战：团队成员沟通效率低，经常出现理解偏差

**预期表现**：
1. 开场白引用"团队成员沟通效率低"
2. Goal阶段问短期目标和时间框架
3. Reality阶段聚焦可控因素和现有资源
4. Options阶段提及沟通技巧和工具
5. Will阶段要求24-48小时内的具体行动

### 测试用例2：职业发展场景

**用户画像**：
- 角色：核心骨干/资深专家
- 业务线：技术/研发
- 发展目标：希望在3年内成为技术管理者

**预期表现**：
1. 开场白引用"成为技术管理者"
2. Goal阶段深挖价值观和长期愿景
3. Reality阶段探索内在动机和核心优势
4. Options阶段讨论能力发展路径
5. Will阶段设定阶段性里程碑（月/季度）
